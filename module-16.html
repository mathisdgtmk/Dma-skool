CONFIGURATION MANYCHAT
  
Mise en place pas a pas de tes automations Instagram
  
    
1
Connecte Instagram a Manychat
manychat.com → connexion Facebook → autorise l acces Instagram Pro → selectionne ton compte. Ton Instagram doit etre un compte Professionnel ou Createur.
Manychat gratuit jusqu a 1000 contacts
    
2
Cree le flow "Mot cle en commentaire"
Automation → New Flow → Add Trigger → Instagram → Comment on Post → Keyword → entre le mot cle (ex: "INFO", "LIEN", "OUI") → construis le message DM automatique qui part quand quelqu un commente ce mot.
    
3
Cree le flow "Reponse a une story"
New Flow → Add Trigger → Instagram → Story Reply → selectionne ta story ou active pour toutes → configure le DM de qualification automatique.
Active sur tes meilleures stories de vente uniquement
    
4
Ajoute une sequence de relance
Dans ton flow apres le premier message → ajoute un delai de 24h → ajoute un deuxieme message de relance si pas de reponse. Ne jamais laisser un prospect sans relance automatique.
    
5
Teste et publie
Clique Test Flow → simule le declencheur depuis un autre compte → verifie que les messages arrivent → clique Publish. Ton automation est active 24h/24.
  
  
    
⚠️
    
Attention aux limites Meta : Instagram limite le nombre de DMs automatiques. Ne configure pas plus de 3 flows actifs en meme temps au debut. Augmente progressivement pour ne pas faire shadowban ton compte.
  
  
LES SECRETS
  
Ce qui fait la difference entre une automation qui convertit et une qui irrite
  
    
Personnalise avec le prenom
    

Manychat recupere automatiquement le prenom Instagram de chaque personne. Utilise la variable {{first name}} dans tous tes messages.

Un DM qui commence par "Hey Thomas !" convertit 3 fois mieux qu un DM qui commence par "Hey toi !". La personnalisation cree l illusion d une conversation reelle.

  
  
    
Pose une question dans le premier DM
    

Ne commence jamais par envoyer un lien directement. Pose d abord une question pour qualifier et engager la conversation.

Exemple : "Tu decouvres ou t as deja essaye de vendre en ligne ?" Quand ils repondent, tu peux prendre la main et continuer manuellement pour closer. L automation attire — toi tu closes.

  
  
    
Combine automation et closing manuel
    

L automation ouvre la conversation — toi tu la closes manuellement pour les prospects chauds.

Workflow ideal : Automation envoie le premier DM → personne repond → tu reprends la conversation manuellement → closing DM (module 11) → vente.

Ne laisse jamais l automation faire tout le travail sur une vente a plus de 50 euros. Le toucher humain est indispensable pour les gros tickets.

  
  
    
Surveille tes stats chaque semaine
    

Manychat te donne des stats sur chaque flow : nombre de declenchements, taux de reponse, taux de clic.

Optimise ce qui marche le moins : si le taux de reponse est inferieur a 20%, change le premier message. Si le taux de clic est inferieur a 10%, change ton offre ou ton lien. A/B teste deux versions differentes du meme flow.
