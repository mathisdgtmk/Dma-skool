GUIDE COMPLET
  
Configuration pas a pas — choisis ta plateforme
  
            
  
    
      
1
Cree ton compte gratuit
systeme.io → Commencer gratuitement → pas de CB demandee → tu accedes immediatement au dashboard complet. Plan gratuit : 2000 contacts, 3 tunnels, emails illimites.
Zero risque — tu paies seulement si tu depasses 2000 contacts
      
2
Cree ton tunnel de vente
Tunnels → Creer un tunnel → Vendre un produit → choisis un template → customise avec tes textes, couleurs et images. Le tunnel genere automatiquement : page capture + page vente + page confirmation.
      
3
Configure ton produit et le paiement
Ventes → Produits → Nouveau produit → nom, prix, description → connecte Stripe → active le paiement en plusieurs fois si tu veux. Chaque achat declenche automatiquement l acces a ton espace membres.
      
4
Cree ton espace membres
Espaces membres → Creer un espace → ajoute tes modules avec videos, textes, fichiers → configure les acces selon les produits achetes. C est la que tu heberges ta formation complete.
Systeme.io heberge tes videos — pas besoin de YouTube
      
5
Active les emails automatiques
Emails → Campagnes → Creer une campagne → ajoute tes 5 emails de bienvenue espaces de 1 jour → active. Chaque inscription declenche automatiquement la sequence.
    
  
  
    
      
1
Cree ton compte Stan
stan.store → Start Free Trial → 14 jours gratuits sans CB → connecte ton Instagram ou TikTok pour importer ton profil. Ton lien Stan remplace immediatement ton lien en bio.
14 jours gratuits — largement suffisant pour faire ta premiere vente
      
2
Ajoute tes produits
Products → Add Product → choisis le type : Digital Product pour un PDF ou fichier, Course pour une formation, Booking pour des appels coaching → remplis les infos et mets ton prix.
      
3
Personnalise ta boutique
Store → Customize → change les couleurs et la photo de profil. Fond sombre + accents or = look premium instantane. Coherent avec ton branding sur les reseaux.
      
4
Connecte Stripe
Settings → Payment Settings → Connect with Stripe → Stan ne prend aucune commission sur tes ventes. Tout l argent va directement sur ton compte Stripe.
      
5
Active le programme d affiliation
Stan → Affiliate Program → active et definis ton taux de commission (20 a 40% recommande) → tes clients peuvent promouvoir ta formation et gagner une commission. Tu crees une armee de vendeurs gratuitement.
L affiliation peut doubler tes ventes sans effort supplementaire
